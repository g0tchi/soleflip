# Metabase Artifacts

This directory contains all files related to the Metabase integration, including SQL queries for questions, definitions for database views, and JSON models for dashboards.

- `/queries`: Contains `.sql` files used for creating questions in Metabase.
- `/views`: Contains Data Definition Language (DDL) for creating database views that simplify analytics in Metabase.
- `/dashboards`: Contains JSON exports of Metabase dashboards for easy re-importing.
